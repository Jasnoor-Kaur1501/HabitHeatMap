# HabitHeatmap

GitHub's contribution graph, but for any habit you want to track.

## Live Demo

🔗 [jasnoor-kaur1501.github.io/HabitHeatmap](https://jasnoor-kaur1501.github.io/HabitHeatmap)

## What it does

- Create any habit with a name, emoji icon, and color
- Log today in one click — or tap any past day on the heatmap to toggle it
- Full 52-week heatmap showing your entire year at a glance
- Live stats — total days, current streak, best streak, this month's rate
- Best months bar chart (last 6 months)
- Day-of-week breakdown — see which days you're most consistent
- Streak badge in the header when you're on a run
- Supports multiple habits — switch between them instantly
- Persists across sessions via localStorage

## Design

Plus Jakarta Sans + Familjen Grotesk · Warm off-white, clean neutrals · Calm productivity dashboard energy — data-forward but still beautiful. Completely different from Days 01–08.

## Tech

HTML · CSS · Vanilla JS · localStorage · No API · No dependencies · GitHub Pages

## Run locally

Download or clone → open `index.html` in any browser. Done.
